# PythonCodeELMCMC
Python (actually Cython) code to fit electron lifetime for XENON1t. 

Fit_func2 : Functions for initialization, seting data files, calculate trend distributions and so on.
LnLike    : LnLike functions.
ImpurityTrend : Calculatoin evolution of impurity
ElectronlifetimeTrend : Convert impurity to ELifetime
FormPars : Set and initialize parameters
HistorianData : Set and Historian File
MCMC_Tools : Functions to help calculations
